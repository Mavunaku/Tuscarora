## 📅 Recent Bookings
To see a live list of bookings here, install the **Dataview** plugin and use this code block:

```dataview
TABLE startDate as "Arriving", endDate as "Departing", roomName as "Room", guests as "Guests"
FROM "Bookings"
SORT startDate DESC
LIMIT 10
```
